Forms

There are different elements used in forms such as:

  • input:
    This HTML element is used to create interactive controls for web-based forms in order to accept data from the user.



  • label:
    This element defines a label for several form elements. The label element is used to associate a text label with a form input field. The label is used to tell users the value that should be entered in the associated input field. Display inline.
  • select:
    The select element is used to define a dropdown list.
  • option
    The option element is used to define an option that can be selected. And by default the first option is already selected.
  • size:
    It is an attribute used to define number of visible values/options.
    multiple: This is also an attribute used to let the user to have multiple selection.


  • textarea:
    The element defines a multi-line input field (a text area):
    row: Row is an attribute that specifies the visible number of lines in a text area.
    cols: The cols attribute specifies the visible width of a text area.


  • button:
    This element defines a clickable button.

    It is always recommended to use (type) attribute for buttons.
  • fieldset:
    The fieldset element is used to group related data in a form.
    Personalia:




  • legend:
    The legend element defines a caption for the fieldset element. Look at the above example.
  • datalist:
    This element defines a list of pre-defiend option for input. This is intersting.
  • output:
    This element represents the result of a calculation (like one performed by a script).
