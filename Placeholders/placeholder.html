Placeholders

Placeholders can be used to enhance the experience of your application. They are built only with HTML and CSS, meaning you do not need any JavaScript to create them. You will, however, need some custom JavaScript to toggle their visibility. Their appearance, color, and sizing can be easily customized with our utility classes. The placeholder is created with (.placeholder) class.

.placeholder & .placeholder-glow

placeholder is created using the .placeholder class and .placeholder-glow and placeholderwave are used to animate the placeholder.

Width

We can change the width through using grid column classes, width utilities or inline style.

Color

By default, the placeholder uses currentColor. This can be overridden with a custom color or utility class.

Sizing

The size of .placeholders are based on the typographic style of the parent element. We can customize them with sizing modifiers such as:

  • .placeholder-lg
  • .placeholder-sm
  • .placeholder-xs

Animation

The placeholder is animated using two following classes:

  • .placeholder-glow
  • .placeholder-wave
